# ZeroMQ Examples #

I gather here ZeroMQ examples to show off some of its patterns, and how
to use them for specific purposes.


# Dependencies #

This examples uses [ZMQ::LibZMQ3][libzmq3pm] and have been tested with
libzmq 3.2.1-rc1.
